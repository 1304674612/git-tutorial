## 创建步骤

1. 访问: https://github.com/new
2. 填入上述信息
3. **不要勾选**: Add a README file, Add .gitignore, Choose a license
4. 点击 "Create repository"

## 创建完成后

仓库地址将是: https://github.com/Mickl/git-tutorial

然后运行以下命令推送代码:
```bash
git push --set-upstream origin main
```
